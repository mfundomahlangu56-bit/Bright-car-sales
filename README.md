# Project Overview
The car sales dataset provides detailed daily transactional records for vehicles sold across all dealership branches.The dataset is designed to support sales analysis,pricing strategy,performance tracking

# Dataset Descrption
--Transactional Details
sale date - timestamp of the transaction
Day of week - derived from the sale date for weekly trand analysis
Transaction Id - unique identifier for each sale
# Vehicle Information 
- Make
- Model
- Mileage
- Year
# Objectives of the Dateset
- Analyze daily sales performance
- Identify revenue trends by day, model and vehicle
-  Understand customer demand across makes and models
